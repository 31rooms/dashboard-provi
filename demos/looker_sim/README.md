# Simulador de Looker Studio - Grupo Provi

Este es un ejemplo estático que simula la interfaz y el comportamiento de un dashboard de Looker Studio para el seguimiento de leads y ventas.

## Mejoras Implementadas
- **Aislamiento**: Movido a la carpeta `demos/looker_sim/` para no interferir con el código fuente del proyecto.
- **Corrección de Diseño**: Se solucionó el problema de crecimiento infinito en los gráficos mediante el uso de contenedores con altura definida y un scroll interno.
- **Pestañas Funcionales**: Se implementó un sistema de pestañas (Resumen, Marketing, Ventas, Operaciones) para organizar la información.
- **Responsividad**: Mejorado para que los gráficos se ajusten correctamente al redimensionar la ventana.

## Uso
Simplemente abre `simulacion_looker.html` en cualquier navegador web.
